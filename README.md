## **📦 사용법**

1. git clone을 이용하여 다운받기

```
git clone https://github.com/igdev116/next-ts-eslint-prettier.git
```

2. yarn을 통한 설치

```
yarn install
```

3. 프로젝트 실행하기

```
yarn dev
```

4. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```
rm -rf .git
```
